For your Information

Information about UMW Reviews
UMW Reviews was a website developed by Bryan Nosar, Kyle Tracy, Sheldon McClung and Shane Chamberlain for CPSC 350

We chose to create this website because Bryan and Sheldon have experience with Data Mining and Kyle had general web development experience. Lastly, Shane joined the group to help with debugging and editing.

Features

Currently the website is still in production. We have made a lot of progress on the website since Bryan had to parse the MovieLens dataset and Sheldon had to retrofit Python Code Users are able to register and log in to the website. Anyone is able to search our database for movies, but only registered users can rate and review movies. Registered users are also allowed to get a list of 15 movies to be recommended based off of their ratings of movies. The newest addition is a language filter. If a user tries to use profanity in a movie review then their review is not added and they are warned about their language usage.

Future of this Site

We currently are in the process of adding additional admin privileges. Admins will be able to ban or delete users and add new movies.

We have error messages are already made on the backend, we just need to use Javascript to display the errors.

We are going to clean the site up a bit more to make it more mobile friendly

This information on the sidebar will probably be made into Modals to have cleaner code.
